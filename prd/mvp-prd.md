# MVP PRD - 线下德州扑克实时辅助

## 1. 功能目标

构建一个手机可用的实时辅助工具，支持你在牌局中录入手牌、位置、筹码、公牌与行动，并输出：

- 成牌概率（牌型完成概率）
- 胜率估计（equity / win rate）
- 下注建议（附解释）

同时覆盖翻前与翻后（flop / turn / river）。

## 2. 用户场景

- 现场边玩边输入，输入速度要求高
- 玩家人数不固定（2-10）
- 有固定盲注（默认 SB/BB = 100/100），但需可自定义
- 玩家可能补码，补码数量手工输入
- 行为不完全标准化，需要自然语言解析

## 3. 范围（第一阶段必须满足）

1. 牌桌配置
   - 人数自定义（2-10）
   - 位置自定义映射到玩家
   - 盲注可配置（默认 100/100）
2. 玩家管理
   - 每场自定义玩家 ID
   - 支持保存玩家画像（风格描述）
   - 支持保存历史行为统计（可信度上下文）
3. 行动录入
   - 快捷按钮：`fold/check/call/raise/all-in`
   - 默认 raise 倍率（2x）+ 自定义筹码输入
   - 自然语言输入并结构化为行动事件
4. 牌局进程
   - preflop / flop / turn / river 状态机
   - 手牌、公牌、底池、有效筹码实时更新
5. 决策建议
   - 主引擎：近似实时
   - 备引擎：稍慢但更准确
   - 输出建议动作 + 原因解释（位置、赔率、范围、读人）

## 4. 非功能要求

- 手机端操作优先（单手、快速点击）
- 响应快：主引擎在 1-2 秒内给出建议
- 成本可控：默认纯本地可运行，不强制服务器
- 可扩展：后续可接入 MiniMax 做自然语言解析增强

## 5. 技术策略（高层）

- 前端：Web App（可后续封装小程序/壳）
- 存储：本地优先（IndexedDB / localStorage）
- 算法：
  - 主引擎：启发式 + 快速 Monte Carlo
  - 备引擎：更高采样 Monte Carlo + 范围修正
- NLP：
  - 初版：模板 + 关键词解析
  - 增强：可选接入 MiniMax

## 6. 验收标准

- 能完整录入一手牌从翻前到河牌
- 任意阶段都能给出概率、胜率、建议和解释
- 录入 10 人桌无明显卡顿
- 玩家历史行为可保存并影响建议说明
